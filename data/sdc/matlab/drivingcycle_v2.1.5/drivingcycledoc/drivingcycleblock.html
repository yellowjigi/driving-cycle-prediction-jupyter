Driving Cycle & Discrete Driving Cycle

Simulink blocks representing commonly-used driving cycles.

Library

Driving Cycle (Simulink Block) - available from MATLAB File Exchange.

Description

These blocks produce driving cycle data, as used in many automotive applications. They output displacement, velocity and acceleration, all in SI units. The user can choose from a list of popular cycles.

I created these blocks to help my students and researchers get up and running quickly. They aren't guaranteed as fit for any particular purpose, but I hope you find them useful.

Parameters

Cycle name
This is a drop-down list of popular cycles such as NEDC and WLTP class 3.
 
Repeat cycle forver
This is a checkbox. If ticked, the simulation will continue until its stop time is reached; if not, it will automatically terminate after two cycles.
 
Sample time
This is only available on the discrete version of the block. It is specified in seconds and should work exactly as expected.
 

Outputs

s (m)
Displacement, in SI units.
 
v (m/s)
Velocity, in SI units.
 
a (m/s^2)
Acceleration, in SI units.
